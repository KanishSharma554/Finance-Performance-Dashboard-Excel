# Finance-Performance-Dashboard-Excel
Interactive Excel dashboard for financial performance analysis and insight generation.


## Overview
This project presents an interactive finance dashboard built in Microsoft Excel to analyze revenue, expense, and profit trends.

## Key Features
- KPI cards (Revenue, Expense, Profit)
- Department-wise revenue vs expense comparison
- Monthly profit trend analysis
- Interactive filters (Year, Month, Quarter, Department)

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Visualizations

## Key Insights
- Identified high-expense departments
- Tracked monthly profitability trends
- Evaluated cost distribution patterns
